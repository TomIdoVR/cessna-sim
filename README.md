# Cessna 172 Sim — United States of Hawaii

A browser-based 3D flight simulator built with Three.js (no frameworks, no build tools).  
Single `index.html` file — open in Chrome or deploy to GitHub Pages.

## Play
Open `index.html` in Chrome, or visit the GitHub Pages URL.

## Controls
| Key | Action |
|-----|--------|
| W / ↓ | Pitch up |
| S / ↑ | Pitch down |
| A / ← | Roll left |
| D / → | Roll right |
| Q / E | Rudder left / right |
| = / + | Throttle up |
| - | Throttle down |
| G | Toggle landing gear |
| C | Cycle camera (Chase / Cockpit / Orbit) |
| N | Toggle night mode |
| F | Toggle fog |

## Models folder
Drop optional GLB files in `models/` to replace the procedural geometry:
- `cessna.glb` — full Cessna 172 exterior
- `cockpit.glb` — cockpit interior
